AVAILABLE SCRIPTS
=================

1. activate_env.ps1 (Simple Direct Activation)
----------------------------------------------

Purpose: Simple, direct activation of an existing virtual environment

Key Features:
- Fast and lightweight - Just activates the environment
- Uses absolute path to activation script
- No setup checks - Assumes environment already exists
- Minimal output - Just activation info and success message

Best for: When you know the virtual environment already exists and just want to activate it quickly.

Usage:
.\activate_env.ps1

2. activate_deep_research.ps1 (Full Setup + Activation)
-------------------------------------------------------

Purpose: Complete environment setup and activation

Key Features:
- Creates environment if missing - Automatically creates .venv if it doesn't exist
- Full setup process - Checks project directory, creates venv, installs requirements
- Comprehensive error handling - Validates everything step by step
- Requirements installation - Automatically installs packages from requirements.txt
- Detailed output - Shows all setup steps and paths

Best for: First-time setup or when you want to ensure everything is properly configured.

Usage:
.\activate_deep_research.ps1

COMPARISON TABLE
================

Feature                          | activate_env.ps1    | activate_deep_research.ps1
--------------------------------|---------------------|------------------------------
Speed                           | Fast (2-3 seconds)  | Slower (10-30 seconds)
Setup                           | No setup            | Full setup
Environment Creation            | Assumes exists      | Creates if missing
Requirements Install            | No                  | Yes (if requirements.txt exists)
Error Handling                  | Basic               | Comprehensive
Output                          | Minimal             | Detailed
Use Case                        | Daily activation    | First-time setup

WHEN TO USE WHICH SCRIPT?
=========================

Use activate_env.ps1 when:
- Environment already exists
- You just want to activate quickly
- Daily development work
- You're in a hurry

Use activate_deep_research.ps1 when:
- First time setting up the project
- Environment might be missing
- You want to ensure requirements are installed
- You want detailed setup information
- Troubleshooting environment issues

TROUBLESHOOTING
===============

Common Issues:

1. "Activation script not found"
   - Run activate_deep_research.ps1 to create the environment first

2. "Python not found"
   - Ensure Python is installed and in your PATH

3. "Requirements installation failed"
   - Check if requirements.txt exists in the project directory
   - Verify internet connection for package downloads

Quick Example:

SCRIPT CONTENTS
===============

activate_env.ps1:
-----------------
# Direct Virtual Environment Activator
# This script directly activates the virtual environment using absolute path
# Usage: .\activate_env.ps1

# Absolute path to the virtual environment activation script
$ActivateScript = "C:\Users\sroy5\Study\llm\llm_agents\agents\2_openai\deep_research\.venv\Scripts\Activate.ps1"

# Check if activation script exists
if (-not (Test-Path $ActivateScript)) {
    Write-Host "Activation script not found at: $ActivateScript" -ForegroundColor Red
    Write-Host "Please ensure the virtual environment exists at the specified path." -ForegroundColor Red
    exit 1
}

# Set environment variables
$env:DEEP_RESEARCH_ROOT = "C:\Users\sroy5\Study\llm\llm_agents\agents\2_openai\deep_research"
$env:PYTHONPATH = $env:DEEP_RESEARCH_ROOT

# Display activation info
Write-Host "=== Activating Deep Research Environment ===" -ForegroundColor Cyan
Write-Host "Activation Script: $ActivateScript" -ForegroundColor White
Write-Host "Project Root: $env:DEEP_RESEARCH_ROOT" -ForegroundColor White
Write-Host "=============================================" -ForegroundColor Cyan

# Activate the virtual environment
Write-Host "Activating virtual environment..." -ForegroundColor Green
& $ActivateScript

if ($LASTEXITCODE -ne 0) {
    Write-Host "Failed to activate virtual environment." -ForegroundColor Red
    exit 1
}

Write-Host "Environment activated successfully!" -ForegroundColor Green
Write-Host "You can now run your deep research scripts." -ForegroundColor White

activate_deep_research.ps1:
---------------------------
# Global Deep Research Environment Activator
# This script can be called from anywhere in the filesystem
# Usage: .\activate_deep_research.ps1

# Define the absolute path to your deep research project
# Update this path to match your actual project location
$DeepResearchPath = "C:\Users\sroy5\Study\llm\llm_agents\agents\2_openai\deep_research"

# Check if the project directory exists
if (-not (Test-Path $DeepResearchPath)) {
    Write-Host "Deep Research project not found at: $DeepResearchPath" -ForegroundColor Red
    Write-Host "Please update the `$DeepResearchPath variable in this script to point to your project location." -ForegroundColor Yellow
    exit 1
}

# Set the absolute path to the virtual environment
$VenvPath = "C:\Users\sroy5\Study\llm\llm_agents\agents\2_openai\deep_research\.venv"
$ActivateScript = "C:\Users\sroy5\Study\llm\llm_agents\agents\2_openai\deep_research\.venv\Scripts\Activate.ps1"

# Check if virtual environment exists
if (-not (Test-Path $VenvPath)) {
    Write-Host "Virtual environment not found at: $VenvPath" -ForegroundColor Yellow
    Write-Host "Creating virtual environment..." -ForegroundColor Green
    
    # Change to project directory
    Push-Location $DeepResearchPath
    
    # Create virtual environment in the project directory
    python -m venv ".venv"
    
    if ($LASTEXITCODE -ne 0) {
        Write-Host "Failed to create virtual environment. Please ensure Python is installed." -ForegroundColor Red
        Pop-Location
        exit 1
    }
    
    Write-Host "Virtual environment created successfully!" -ForegroundColor Green
    Pop-Location
}

# Check if activation script exists
if (-not (Test-Path $ActivateScript)) {
    Write-Host "Activation script not found at: $ActivateScript" -ForegroundColor Red
    Write-Host "Please ensure the virtual environment is properly created." -ForegroundColor Red
    exit 1
}

# Set environment variables
$env:DEEP_RESEARCH_ROOT = $DeepResearchPath
$env:PYTHONPATH = $DeepResearchPath

# Display current setup
Write-Host "=== Deep Research Environment Setup ===" -ForegroundColor Cyan
Write-Host "Project Root: $DeepResearchPath" -ForegroundColor White
Write-Host "Virtual Environment: $VenvPath" -ForegroundColor White
Write-Host "Activation Script: $ActivateScript" -ForegroundColor White
Write-Host "Python Path: $env:PYTHONPATH" -ForegroundColor White
Write-Host "=======================================" -ForegroundColor Cyan

# Activate the virtual environment using the absolute path
Write-Host "Activating virtual environment..." -ForegroundColor Green
& $ActivateScript

if ($LASTEXITCODE -ne 0) {
    Write-Host "Failed to activate virtual environment." -ForegroundColor Red
    exit 1
}

Write-Host "Environment setup complete!" -ForegroundColor Green
Write-Host "You can now run your deep research scripts." -ForegroundColor White

# Optional: Install requirements if requirements.txt exists
$RequirementsFile = Join-Path $DeepResearchPath "requirements.txt"
if (Test-Path $RequirementsFile) {
    Write-Host "Installing requirements from requirements.txt..." -ForegroundColor Green
    pip install -r $RequirementsFile
    
    if ($LASTEXITCODE -ne 0) {
        Write-Host "Warning: Some requirements may not have installed correctly." -ForegroundColor Yellow
    } else {
        Write-Host "Requirements installed successfully!" -ForegroundColor Green
    }
}

Write-Host "`nEnvironment activated successfully!" -ForegroundColor Green
Write-Host "Virtual environment location: $VenvPath" -ForegroundColor White
Write-Host "Activation script: $ActivateScript" -ForegroundColor White
Write-Host "You can now run your deep research scripts from anywhere." -ForegroundColor White
